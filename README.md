# 🏏 International T20 Cricket Data Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on International T20 cricket match data using Python.
The objective is to explore the dataset, clean the data, perform statistical analysis, and create meaningful visualizations to understand patterns in international T20 matches.

The analysis is implemented in a **Jupyter Notebook using Google Colab**, making it easy to reproduce and understand.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Apply **Pandas and NumPy** functions
* Conduct statistical analysis
* Create data visualizations
* Identify trends and patterns in international T20 matches

---

## 🗂 Dataset

The dataset used in this project:

**International_T20_Data.csv**

It contains information such as:

* Match venues
* Cities
* Match outcomes
* Winning teams
* Run margins
* Wicket margins
* Toss decisions
* Match dates
* Match gender category

## 📊 Data Analysis Workflow

### 1️⃣ Data Loading

* Load dataset using **Pandas**

### 2️⃣ Data Exploration

* Check dataset shape
* Inspect columns
* Analyze data types
* Generate statistical summary

### 3️⃣ Data Cleaning

* Handle missing values
* Remove duplicates
* Rename columns
* Create new features

### 4️⃣ Statistical Analysis

Using **NumPy** functions:

* Mean
* Median
* Standard deviation
* Variance
* Maximum & minimum values

### 5️⃣ Data Aggregation

Using **Pandas operations**:

* `groupby`
* `value_counts`

* Bar charts

* Box plots
